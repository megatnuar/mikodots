This is my dotfiles for Arch linux based distros. Have tried with a clean Arch install and Endeavour OS. The latter works best.

To install, copy the dotfiles into your home directory and run:

```
cd scripts
./install_packages.sh
```

Then,

```
cd ..
stow .
```

Enjoy!
